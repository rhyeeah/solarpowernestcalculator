🔥 Firebase Setup Guide

Sundin ang mga hakbang na ito para i-activate ang auth + lead saving:

  1. Pumunta sa console.firebase.google.com → Create a new project → pangalanan itong solar-power-nest
  2. Sa Project Settings → Your Apps → click </> Web → i-register ang app → i-copy ang firebaseConfig object
  3. I-paste ang config sa <script type="module"> sa simula ng HTML file (hanapin ang YOUR_API_KEY)
  4. Sa Firebase Console → Authentication → Sign-in method → I-enable ang Google
  5. Sa Firebase Console → Firestore Database → Create database → Start in production mode
  6. Sa Firestore → Rules → i-paste ito:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /leads/{doc} { allow read, write: if request.auth != null; }
        match /payments/{doc} { allow read, write: if request.auth != null; }
        match /users/{uid} { allow read, write: if request.auth != null; }
      }
    }
  7. I-update ang ADMIN_EMAIL sa script sa inyong Google email para makita ang Leads Admin tab
  8. I-deploy ang HTML sa Firebase Hosting o GitHub Pages — tapos na! 🎉
